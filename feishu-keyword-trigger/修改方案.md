> 文档生成时间：2026-03-06

---

## 📋 功能需求

当前行为：
- 群聊中必须 @机器人 才会触发回复
- 配置 `requireMention: false` 后仍然无效

期望行为：
- 群聊中消息包含关键词（如"傻妞"）即可触发回复
- 无需 @机器人
- 可自定义关键词列表

({ requireMention } = resolveFeishuReplyPolicy({

if (requireMention && !ctx.mentionedBot) {

**修改后代码：**
```typescript
({ requireMention } = resolveFeishuReplyPolicy({
  isDirectMessage: false,
  globalConfig: feishuCfg,
  groupConfig,
}));

// --- 关键词触发功能 ---
// 检查消息是否包含触发关键词（无需@机器人）
const triggerKeywords = feishuCfg?.triggerKeywords || [];
const hasTriggerKeyword = triggerKeywords.length > 0 && 
  triggerKeywords.some((kw: string) => ctx.content.includes(kw));

if (requireMention && !ctx.mentionedBot && !hasTriggerKeyword) {
  log(`feishu[${account.accountId}]: message in group ${ctx.chatId} did not mention bot and no trigger keyword`);
  // Record to pending history for non-broadcast groups only. For broadcast groups,
  // the mentioned handler's broadcast dispatch writes the turn directly into all
  // agent sessions — buffering here would cause duplicate replay when this account
  // later becomes active via buildPendingHistoryContextFromMap.
  if (!broadcastAgents && chatHistories && groupHistoryKey) {
    recordPendingHistoryEntryIfEnabled({
      historyMap: chatHistories,
      historyKey: groupHistoryKey,
      limit: historyLimit,
      entry: {
        sender: ctx.senderOpenId,
        body: `${ctx.senderName ?? ctx.senderOpenId}: ${ctx.content}`,
        timestamp: Date.now(),
        messageId: ctx.messageId,
      },
    });
  }
  return;
}
// --- 关键词触发功能结束 ---
```

**修改说明：**
- 新增 `triggerKeywords` 配置读取
- 新增 `hasTriggerKeyword` 判断逻辑
- 修改条件判断：增加 `&& !hasTriggerKeyword`
