## 📋 配置说明

### 全局配置（所有群组）
```json
{
  "channels": {
    "feishu": {
      "triggerKeywords": ["傻妞", "傻妞在吗", "叫傻妞"]
    }
  }
}
```

### 群组单独配置
```json
{
  "channels": {
    "feishu": {
      "groups": {
        "oc_xxx": {
          "requireMention": false,
          "triggerKeywords": ["傻妞", "AI", "助手"]
        }
      }
    }
  }
}
```

## ⚠️ 注意事项

1. **OpenClaw 更新后** - 修改会被覆盖，需要重新运行 `apply-patch.ps1`
2. **备份很重要** - 回滚脚本依赖备份文件，不要删除
3. **调试日志** - 运行 `openclaw logs --follow` 查看详细日志

## 🆘 遇到问题？

### 问题 1：修改后不生效
- 检查网关是否重启：`openclaw gateway restart`
- 查看日志：`openclaw logs --follow`
- 确认配置格式正确（JSON 语法）

### 问题 2：傻妞回复异常
- 运行回滚脚本恢复原状
- 检查是否有其他配置冲突

### 问题 3：OpenClaw 更新后
- 重新运行 `apply-patch.ps1`
- 或等待官方支持此功能
